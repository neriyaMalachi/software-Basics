נסה בעצמך
גיל:
הזן גיל ולחץ בדוק
ההזחה (Indentation) — חשוב מאוד!
ב-Python, הרווחים (4 רווחים או Tab) בתחילת השורה הם חלק מהשפה — לא סגנון. הקוד שנמצא בתוך if חייב להיות עם הזחה. בלי זה — שגיאה.
חלק ג׳ — אופרטורי השוואה

איך בודקים תנאים?

סימןמשמעותדוגמה

שים לב
= (שווה אחד) = שמירת ערך במשתנה. == (שווה כפול) = השוואה. זה ההבדל: x = 5 שומר, x == 5 בודק.
חלק ד׳ — if בלי else

else הוא אופציונלי

לפעמים אין צורך ב-else — רוצים רק לעשות משהו אם התנאי נכון, ואם לא — פשוט להמשיך.

if בלי else
score = 95

if score == 100:
    print("ציון מושלם! מדהים!")

לחץ על כל שאלה לתשובה.

1. למה ההזחה (רווחים בהתחלת שורה) חשובה כל כך ב-Python?
לחץ לתשובה
כי ב-Python ההזחה מגדירה מה "בפנים" ומה "בחוץ". הקוד שמוזח (4 רווחים) אחרי if — הוא החלק שרץ רק אם התנאי נכון. ב-Python זה חלק מהשפה עצמה, לא רק עניין של סגנון כתיבה.
2. מה ההבדל בין = לבין ==?
לחץ לתשובה
= אחד הוא השמה — שמים ערך בתוך משתנה (x = 5). == כפול הוא השוואה — בודקים אם שני דברים שווים (if x == 5). זהו אחד הבלבולים הנפוצים ביותר אצל מתחילים.
3. כתוב תנאי שבודק אם מספר גדול מ-100 ומדפיס הודעה מתאימה
לחץ לתשובה
num = int(input("הכנס מספר: ")) — ואז: if num > 100: ← שורה חדשה ← (4 רווחים) print("המספר גדול מ-100!") ← else: ← (4 רווחים) print("המספר לא גדול מ-100")
