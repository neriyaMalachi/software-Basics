## 🧭 על הקורס הזה

הקורס הזה נוצר עבור **אדם אחד ספציפי**: מישהו שרואה קוד ולא מבין מילה. שמעולם לא פתח טרמינל. שחשב שתכנות זה "לא בשבילו".

הוא לא קורס אקדמי. הוא לא מכסה "את כל Python". הוא לא מתחיל עם תיאוריה יבשה על אלגוריתמים.

הוא מתחיל מהשאלה הכי בסיסית שיש:

> **"מה זה בכלל מחשב, ולמה צריך לתת לו הוראות?"**

ומשם — צעד אחד בכל פעם.

## 🗓️ מבנה הקורס

הקורס מחולק ל**4 שלבים**, סך הכל **20 שיעורים**, בקצב של שיעור ביום.

### שלב 0 — מה זה בכלל תכנות? `ימים 1–3`

לפני שנוגעים בקוד — מבינים את העולם שאנחנו נכנסים אליו.

```
יום 1  →  מה זה מחשב ותוכנה?
יום 2  →  מה זה שפת תכנות?
יום 3  →  הכלים: Google Colab + שורה ראשונה
```

---

### שלב 1 — Python בסיסי `ימים 4–10`

הלבנים הבסיסיות של כל שפת תכנות.

```
יום 4   →  משתנים ומספרים          x = 5
יום 5   →  טקסט (strings)          name = "דוד"
יום 6   →  קלט מהמשתמש             input("מה שמך?")
יום 7   →  תנאים: if / else        if age >= 18:
יום 8   →  תנאים מורכבים: elif     and, or, not
יום 9   →  לולאת while             while x < 10:
יום 10  →  לולאת for               for i in range(5):
```

---

### שלב 2 — Python מתקדם `ימים 11–16`

מבנים חזקים יותר שמאפשרים לכתוב תוכניות אמיתיות.

```
יום 11  →  רשימות (lists)          fruits = ["תפוח", "בננה"]
יום 12  →  for + רשימות            לרוץ על כל פריט
יום 13  →  פונקציות (def)          def greet(name):
יום 14  →  מילונים (dicts)         {"שם": "דוד", "גיל": 30}
יום 15  →  שגיאות: try / except    קוד שלא מתרסק
יום 16  →  קריאת קבצים             open("file.txt")
```

---

### שלב 3 — פרויקט ראשון `ימים 17–20`

```
יום 17  →  חזרה כללית + חידון
יום 18  →  תכנון הפרויקט
יום 19  →  כתיבת הקוד
יום 20  →  סיום + מה הלאה?
```
