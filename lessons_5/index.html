טקסט — Strings

לא רק מספרים — Python עובדת מצוין גם עם מילים, משפטים, וכל סוג של טקסט.

Python 15 דקות קוד ב-Colab
חלק א׳ — מה זה String?

String = שרשרת תווים

בשיעור הקודם עבדנו עם מספרים. אבל תוכניות אמיתיות צריכות גם לעבוד עם טקסט — שמות, הודעות, כתובות. בפייתון, טקסט נקרא String.

String = כל דבר שעטוף במרכאות. יכול להיות אות אחת, מילה, משפט שלם, או אפילו ריק לגמרי.

המחרוזת "שלום" — כל תו יושב במקום מוגדר:
ש
0
ל
1
ו
2
ם
3
יצירת strings
name = "דוד"
city = "תל אביב"
greeting = "שלום עולם!"

print(name)      # דוד
print(city)      # תל אביב
print(greeting)  # שלום עולם!
חלק ב׳ — חיבור טקסטים

לחבר strings עם +

בדיוק כמו שמחברים מספרים, אפשר לחבר strings. הפלט יהיה הטקסט המחובר.

חיבור טקסטים
first = "שלום"
second = " עולם"
print(first + second)   # שלום עולם

name = "רחל"
print("שלום, " + name + "!")  # שלום, רחל!
שגיאה נפוצה
אי אפשר לחבר ישירות מספר עם string: "גיל: " + 25 יגרום לשגיאה. צריך להמיר את המספר לטקסט קודם: "גיל: " + str(25) — או להשתמש בפסיק בתוך print: print("גיל:", 25)
חלק ג׳ — פעולות שימושיות

מה אפשר לעשות עם string?

len()
מספר התווים
len("שלום") → 4
.upper()
הכל לאותיות גדולות
"hello".upper() → "HELLO"
.lower()
הכל לאותיות קטנות
"HELLO".lower() → "hello"
.replace()
מחליף טקסט בטקסט
"שלום".replace("ש","כ") → "כלום"
.strip()
מסיר רווחים בקצוות
" hi ".strip() → "hi"
f-string
דרך נוחה לשלב משתנים
f"שלום {name}" → "שלום דוד"
f-string — הדרך המודרנית והנוחה
name = "דוד"
age = 30

print(f"שלום, {name}! אתה בן {age}.")
# יציג: שלום, דוד! אתה בן 30.
f-string הוא כלי עוצמתי
שים f לפני המרכאות, ואז כתוב את שם המשתנה בתוך סוגריים מסולסלים {}. Python תחליף אוטומטית את השם בערך שלו. זה הרבה יותר נוח מחיבור עם +.

לחץ על כל שאלה לתשובה.

1. מה ההבדל בין המספר 5 ל-string "5"?
לחץ לתשובה
המספר 5 הוא מספר — אפשר לחשב איתו (5 + 3 = 8). ה-string "5" הוא טקסט — אם תחבר "5" + "3" תקבל "53" ולא 8! זו אחת השגיאות הנפוצות ביותר.
2. מה יציג: print(len("Python"))?
לחץ לתשובה
יציג 6 — כי המילה "Python" מכילה 6 תווים: P, y, t, h, o, n.
3. כתוב שורת קוד שמדפיסה "שלום, [שמך]! ברוך הבא." תוך שימוש ב-f-string
לחץ לתשובה
name = "ישראל" — ואז: print(f"שלום, {name}! ברוך הבא.") — הסוגריים המסולסלים {} אומרים ל-Python: "החלף את זה בערך המשתנה".
