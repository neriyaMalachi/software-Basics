לולאת while

מה אם צריך לעשות אותו דבר שוב ושוב, עד שמשהו משתנה? לזה יש לולאה.

Python 15 דקות קוד ב-Colab
חלק א׳ — מה זה לולאה?

לחזור על אותה פעולה — אוטומטית

דמיין שצריך לספור מ-1 עד 10. בלי לולאה, היית צריך לכתוב 10 שורות print. עם לולאה — שורה אחת עושה הכל.

לולאת while אומרת: "כל עוד התנאי הזה נכון — המשך לחזור."

כך Python מריצה while x < 4
1
בדיקה: x < 4? (x=1) → כן
print(x) → מדפיס 1 | x = x + 1 → x=2
2
בדיקה: x < 4? (x=2) → כן
print(x) → מדפיס 2 | x = x + 1 → x=3
3
בדיקה: x < 4? (x=3) → כן
print(x) → מדפיס 3 | x = x + 1 → x=4
✕
בדיקה: x < 4? (x=4) → לא → עוצרים
חלק ב׳ — תחביר

איך כותבים while

ספירה מ-1 עד 5
x = 1

while x <= 5:
    print(x)
    x = x + 1  # חשוב! בלי זה — לולאה אינסופית

print("סיימנו!")

# פלט: 1  2  3  4  5  סיימנו!
סכנה — לולאה אינסופית!
אם שוכחים לשנות את המשתנה שבתנאי — הלולאה רצה לנצח. המחשב לא יעצור לבד. ב-Colab: לחץ על ■ (Stop) כדי לעצור. תמיד בדוק: "האם x משתנה בכל סיבוב?"

חלק ד׳ — דוגמאות שימושיות

while בפעולה אמיתית

משחק ניחוש — while עד שמנחשים נכון
secret = 7
guess = 0

while guess != secret:
    guess = int(input("נחש מספר בין 1 ל-10: "))
    if guess < secret:
        print("גבוה יותר!")
    elif guess > secret:
        print("נמוך יותר!")

print("כל הכבוד, ניחשת נכון!")
סכימת מספרים — while עם מצבר
total = 0
num = 1

while num <= 10:
    total = total + num
    num += 1

print(f"סכום 1 עד 10 = {total}")
# פלט: סכום 1 עד 10 = 55
break — לעצור לולאה מבפנים
לפעמים רוצים לעצור לפני שהתנאי הסתיים. המילה break עוצרת את הלולאה מיד:

while True:
    name = input("שם: ")
    if name != "": break
תרגיל בדוק את עצמך

לחץ על כל שאלה לתשובה.

1. מה חייב לקרות בתוך כל לולאת while כדי שלא תהיה אינסופית?
לחץ לתשובה
המשתנה שנמצא בתנאי חייב להשתנות בכל סיבוב — כדי שבשלב מסוים התנאי יהפוך ל-False והלולאה תעצור. אם x < 10 הוא התנאי, אז x חייב לגדול בכל סיבוב.
2. כמה פעמים תרוץ הלולאה: x=1, while x < 5: x += 1?
לחץ לתשובה
4 פעמים. x מתחיל ב-1 ומסתיים ב-4 (כשx=5 התנאי נכשל). הסיבובים: x=1, x=2, x=3, x=4 — ואז x=5 ועוצרים.
3. מה עושה break?
לחץ לתשובה
break עוצר את הלולאה מיד — גם אם התנאי עדיין True. שימושי כשרוצים לצאת מלולאה לפי תנאי פנימי, בלי לחכות שהתנאי החיצוני ישתנה.
