fruits = ["תפוח", "בננה", "תפוז"]

print(fruits[0])   # תפוח — הראשון
print(fruits[1])   # בננה — השני
print(fruits[-1])  # תפוז — האחרון (-1 = מהסוף)
print(len(fruits)) # 3 — כמות הפריטים
Python סופרת מ-0
הפריט הראשון הוא [0], השני [1], השלישי [2] — וכן הלאה. ו-[-1] מגיע מהסוף — תמיד נותן את האחרון, ללא קשר לאורך הרשימה.

לחץ על כל שאלה לתשובה.

1. מה ההבדל בין fruits[0] ל-fruits[-1]?
לחץ לתשובה
fruits[0] הוא הפריט הראשון. fruits[-1] הוא הפריט האחרון — Python מאפשרת אינדקס שלילי שסופר מהסוף. שימושי כשלא יודעים את אורך הרשימה.
2. מה ההבדל בין remove() ל-pop()?
לחץ לתשובה
remove(x) מסיר לפי ערך — צריך לדעת מה הפריט. pop() מסיר את האחרון ומחזיר אותו — שימושי כשרוצים לשמור את הפריט שהוסר. pop() יכול לקבל גם אינדקס: pop(0) מסיר את הראשון.
3. יש רשימה: nums = [3, 1, 4, 1, 5]. מה יציג max(nums)?
לחץ לתשובה
יציג 5 — כי max() מחזיר את הערך הגדול ביותר ברשימה. sum(nums) יציג 14, min(nums) יציג 1, ו-len(nums) יציג 5.
