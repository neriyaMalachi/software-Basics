for + רשימות

שילוב הלולאה עם הרשימה — אחד הכלים הכי שימושיים בכל תכנות.

שלב 2 15 דקות קוד ב-Colab
חלק א׳ — לרוץ על רשימה

for item in list — בדיוק כך

כשמחברים for עם רשימה, הלולאה עוברת על כל פריט — אחד אחד, בסדר. זו אחת הפעולות הנפוצות ביותר ב-Python.

מעבר על כל פריט ברשימה
fruits = ["תפוח", "בננה", "תפוז", "ענב"]

for fruit in fruits:
    print(f"אני אוהב {fruit}")

# פלט:
# אני אוהב תפוח
# אני אוהב בננה
# אני אוהב תפוז
# אני אוהב ענב
מה קורה בכל סיבוב
בכל סיבוב, המשתנה fruit מקבל את הערך של הפריט הנוכחי. בסיבוב 1 הוא "תפוח", בסיבוב 2 "בננה", וכן הלאה — עד שנגמרו הפריטים.
חלק ב׳ — חישובים על רשימה

ממוצע ציונים — תרגיל קלאסי

חישוב ממוצע
grades = [85, 92, 78, 95, 88]

# דרך 1 — with for loop
total = 0
for grade in grades:
    total += grade
avg = total / len(grades)
print(f"ממוצע: {avg}")

# דרך 2 — קצרה יותר עם sum()
avg = sum(grades) / len(grades)
print(f"ממוצע: {avg}")  # 87.6
חלק ג׳ — סינון פריטים

for + if — שילוב עוצמתי

אחד הדפוסים הנפוצים ביותר: לעבור על רשימה ולסנן רק את הפריטים שעומדים בתנאי.

סינון — רק ציונים מעל 80
grades = [72, 85, 60, 91, 78, 95, 55]
high_grades = []

for grade in grades:
    if grade > 80:
        high_grades.append(grade)

print(high_grades)  # [85, 91, 95]
print(f"{len(high_grades)} תלמידים עברו 80")
חלק ד׳ — נסה בעצמך

מחשבון ציונים אינטראקטיבי

הוסף ציונים וחשב סטטיסטיקות
הוסף ציונים ולחץ "חשב ממוצע"
דפוס שתמיד יחזור
רשימה ריקה → לולאת for → if בתוך הלולאה → append לרשימה חדשה. זה נקרא "filtering pattern" וזה אחד הדפוסים הנפוצים ביותר בתכנות אמיתי.
תרגילבדוק את עצמך

לחץ על כל שאלה לתשובה.

1. מה ההבדל בין for i in range(5) לבין for item in my_list?
לחץ לתשובה
range(5) יוצר מספרים (0,1,2,3,4) — i מקבל מספר בכל סיבוב. my_list עוברת על הפריטים עצמם — item מקבל את הפריט האמיתי. לרשימת פירות: for fruit in fruits נותן את שם הפרי, לא את האינדקס שלו.
2. כיצד אפשר לקבל גם את האינדקס וגם את הפריט בלולאה?
לחץ לתשובה
משתמשים ב-enumerate(): for i, fruit in enumerate(fruits): — אז i הוא האינדקס (0,1,2...) ו-fruit הוא הפריט. שימושי כשצריך לדעת "הפריט ה-3 הוא..."
3. כתוב קוד שמונה כמה מספרים זוגיים יש ברשימה [1,2,3,4,5,6,7,8]
לחץ לתשובה
count = 0 ← nums = [1,2,3,4,5,6,7,8] ← for n in nums: ← (4 רווחים) if n % 2 == 0: ← (8 רווחים) count += 1 ← print(count). התשובה היא 4 (המספרים 2,4,6,8). % הוא שארית — זוגי = שארית 0 בחילוק ב-2.
