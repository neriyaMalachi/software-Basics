מחשבים לא מבינים עברית (ולא אנגלית)

בשיעור הקודם למדנו שמחשב מבצע הוראות. אבל יש בעיה אחת קטנה: המחשב לא מבין שפת אנוש. הוא מבין רק שפת מכונה — רצפים של אפסים ואחדות.

לדוגמה, הפקודה "הדפס את המילה שלום" נראית כך בשפת מכונה:

שפת תכנות = מתרגם בין אדם למחשב

שפת תכנות היא שפה שנמצאת באמצע — קרובה מספיק לשפת אדם כדי שנוכל לכתוב אותה, אבל מוגדרת מספיק כדי שהמחשב יוכל לתרגם אותה לאפסים ואחדות.

דוגמה — איך מוסרים הוראה למחשב
מחשבה אנושית
הדפס על המסך את המשפט:
"שלום עולם"
→
Python
print("שלום עולם")

שורת הקוד print("שלום עולם") — זה כל מה שצריך לכתוב. תוכנה שנקראת מפרש (interpreter) מתרגמת אותה לשפת מכונה ברגע שמריצים.

אנלוגיה
דמיין שאתה בפריז ולא יודע צרפתית. אתה אומר למתורגמן שלך "בקש את החשבון". הוא מתרגם ל-"L'addition, s'il vous plaît". שפת התכנות היא המתורגמן הזה.

למה Python קודם?
Python היא הפשוטה ביותר ללמידה — הקוד שלה נקרא כמעט כמו אנגלית רגילה. אחרי שתבין את הרעיונות הבסיסיים ב-Python, יהיה הרבה יותר קל ללמוד כל שפה אחרת. הרעיונות זהים — רק התחביר משתנה.

תרגיל בדוק את עצמך

לחץ על כל שאלה כדי לראות את התשובה.

1. למה בני אדם לא כותבים ישירות בשפת מכונה?
לחץ לתשובה
כי שפת מכונה היא רצפים של אפסים ואחדות שאי אפשר לקרוא, לכתוב, או להבין בתור אדם. שפות תכנות נוצרו כדי לתת לנו שפה שאנחנו יכולים לכתוב, ותוכנה תתרגם אותה למכונה.
2. מה עושה "מפרש" (interpreter)?
לחץ לתשובה
המפרש הוא תוכנה שקוראת את הקוד שכתבת (למשל ב-Python) ומתרגמת אותו לשפת מכונה שהמחשב מבין — בזמן אמת, שורה אחר שורה.
3. אם לומדים Python היום, האם אפשר ללמוד JavaScript מחר בקלות?
לחץ לתשובה
כן! הרעיונות הבסיסיים — משתנים, תנאים, לולאות, פונקציות — קיימים בכל שפת תכנות. ברגע שמבינים אותם ב-Python, ללמוד שפה חדשה זה בעיקר ללמוד תחביר שונה, לא רעיונות חדשים לחלוטין.
4. באיזו שפה משתמשים לבניית אתרים?
לחץ לתשובה
בעיקר JavaScript — היא שפת הדפדפן. גם HTML ו-CSS מעורבים (אבל הם לא שפות תכנות ממש). Python יכולה לשמש לחלק ה"אחורי" של האתר (server-side), אבל JavaScript שולטת בצד הנראה למשתמש.
