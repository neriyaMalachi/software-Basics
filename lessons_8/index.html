תנאים מורכבים — elif

מה עושים כשיש יותר משתי אפשרויות? ואיך משלבים כמה תנאים ביחד?

Python 15 דקות קוד ב-Colab
חלק א׳ — הבעיה עם if/else

מה אם יש יותר מ-2 אפשרויות?

בשיעור הקודם למדנו if/else — כן או לא. אבל החיים יותר מורכבים מכך. ציון יכול להיות A, B, C, D או F — לא רק "עבר" או "נכשל".

לכן יש elif — קיצור של "else if". זה מאפשר לנו לבדוק אפשרויות נוספות לפני ה-else האחרון.

A
90–100
B
80–89
C
70–79
D
60–69
F
0–59
חלק ב׳ — elif בפעולה

מסיים ציון לאות

ציון → אות — הרץ ב-Colab
score = int(input("מה הציון שלך (0-100)? "))

if score >= 90:
    print("A — מצוין!")
elif score >= 80:
    print("B — טוב מאוד")
elif score >= 70:
    print("C — טוב")
elif score >= 60:
    print("D — עובר")
else:
    print("F — נכשל")
נסה — הכנס ציון
ציון:
הזן ציון ולחץ בדוק
איך Python בודקת?
Python בודקת את התנאים מלמעלה למטה — ועוצרת ברגע שמצאה תנאי נכון. לכן סדר ה-elif חשוב! אם היינו כותבים score >= 60 ראשון — כל ציון מ-60 ומעלה היה מקבל D.
חלק ג׳ — and, or, not

לשלב כמה תנאים ביחד

לפעמים צריך לבדוק שתי דברים בו-זמנית. לשם כך יש מילות קשר:

and
גם וגם — שניהם חייבים להיות נכון
age >= 18 and has_id
or
אחד לפחות — מספיק שאחד נכון
is_student or is_senior
not
הפוך — הופך נכון ללא נכון
not is_banned
דוגמאות עם and ו-or
age = 25
has_ticket = True

# and — שניהם חייבים להיות נכון
if age >= 18 and has_ticket:
    print("כניסה מותרת!")

# or — מספיק שאחד נכון
is_student = True
is_senior = False

if is_student or is_senior:
    print("זכאי להנחה!")
חלק ד׳ — דוגמה מורכבת

בודק BMI — שילוב הכל

מחשבון BMI פשוט
weight = float(input("משקל בק״ג: "))
height = float(input("גובה במטרים (לדוגמה 1.75): "))

bmi = weight / (height ** 2)
print(f"ה-BMI שלך: {bmi:.1f}")

if bmi < 18.5:
    print("תת משקל")
elif bmi < 25:
    print("משקל תקין")
elif bmi < 30:
    print("עודף משקל")
else:
    print("השמנה")
כל הכבוד — זו תוכנה אמיתית!
השתמשת ב-input, המרה למספר עשרוני (float), חישוב, ו-if/elif/else. זה בדיוק מה שתוכניות אמיתיות עושות — רק בקנה מידה גדול יותר.
תרגיל בדוק את עצמך

לחץ על כל שאלה לתשובה.

1. מה ההבדל בין elif ל-else?
לחץ לתשובה
elif (else if) בודק תנאי נוסף — רק אם כל התנאים לפניו לא התקיימו. else לא בודק כלום — הוא פשוט "כל שאר המקרים". אפשר להוסיף כמה elif שרוצים, אבל else תמיד אחרון ויחיד.
2. מה ההבדל בין and ל-or?
לחץ לתשובה
and = גם וגם. שני התנאים חייבים להיות נכון. or = אחד לפחות. מספיק שאחד מהם נכון. דוגמה: "בן 18 AND יש לו כרטיס" — צריך שניהם. "סטודנט OR קשיש" — מספיק אחד.
3. אם score = 75, איזה elif ירוץ בדוגמה של הציון לאות?
לחץ לתשובה
ה-elif של score >= 70 — כי 75 גדול מ-70. Python תבדוק מלמעלה: 75 >= 90? לא. 75 >= 80? לא. 75 >= 70? כן! — ותעצור שם ותדפיס "C — טוב".
4. כמה elif אפשר לשים בין if ל-else?
לחץ לתשובה
כמה שרוצים! אין מגבלה. בפועל, אם יש הרבה מאוד מקרים שונים, מתכנתים לפעמים משתמשים בכלים אחרים (כמו dictionary) — אבל elif עובד תמיד.
