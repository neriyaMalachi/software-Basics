יצירה וגישה למילון
person = {
    "שם": "דוד כהן",
    "גיל": 30,
    "עיר": "תל אביב"
}

print(person["שם"])   # דוד כהן
print(person["גיל"])   # 30

# שינוי ערך
person["גיל"] = 31

# הוספת מפתח חדש
person["מקצוע"] = "מהנדס"
בדיקה אם מפתח קיים
if "גיל" in person: — בודק אם המפתח קיים לפני גישה. אחרת תקבל שגיאה. אפשר גם: person.get("גיל", 0) — אם לא קיים, מחזיר 0 במקום שגיאה.

מעבר על כל זוגות מפתח-ערך
person = {"שם": "דוד", "גיל": 30, "עיר": "תל אביב"}

for key, value in person.items():
    print(f"{key}: {value}")

# שם: דוד
# גיל: 30
# עיר: תל אביב

לחץ על כל שאלה לתשובה.

1. מה ההבדל בין list לבין dict?
לחץ לתשובה
List שומר ערכים בסדר מסוים — גישה לפי אינדקס מספרי (fruits[0]). Dict שומר זוגות מפתח-ערך — גישה לפי שם (person["שם"]). dict מתאים כשלכל פריט יש שם משמעותי, לא רק מיקום.
2. מה קורה אם מנסים לגשת למפתח שלא קיים?
לחץ לתשובה
שגיאת KeyError — Python לא מוצאת את המפתח ונופלת. הפתרון: להשתמש ב-.get() שמחזיר None (או ערך ברירת מחדל) במקום לזרוק שגיאה. person.get("שם", "לא ידוע").
3. כיצד מוסיפים זוג מפתח-ערך חדש למילון קיים?
לחץ לתשובה
פשוט: person["מפתח חדש"] = "ערך חדש". אם המפתח קיים — זה ידרוס אותו. אם לא קיים — זה יוסיף אותו. אין צורך בפונקציה מיוחדת.
